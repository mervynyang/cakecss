cakecss简介

cakecss是基于移动端开发的轻量级material design的css库。压缩后只有15k不到。可通过bower安装。或者直接下载zip文件。

cakecss主要的样式定义为：类名 + 辅助样式 + 辅助样式，比如： <button class="button success large">button</button>。可以多个辅助样式一起使用，也能单独使用。

                
                    bower install onecake

响应式列重置

在某些阈值时，某些列可能会出现比别的列高的情况。用visible-block就可以。

                
                <div class="visible-block"></div>

按钮

按钮的类名是button，其辅助样式有：

颜色类

                
                <button class="button primary">primary</button>
                <button class="button success">success</button>
                <button class="button dark">dark</button>
                <button class="button danger">danger</button>

形状类

                
                <button class="button primary fab">fab</button>
                <button class="button success round">round</button>
                <button class="button danger radius">radius</button>

大小类

                
                <button class="button primary sm">sm</button>
                <button class="button primary">default</button>
                <button class="button primary large">large</button>

状态类

                
                <button class="button primary sm">sm</button>
                <button class="button primary">default</button>
                <button class="button primary large">large</button>

表单

普通表单

                
                <div class="textfield">
                    <input type="text" placeholder="input">
                </div>
                <div class="textfield">
                    <textarea placeholder="textarea"></textarea>
                </div>

带label的表单

                
                <div class="textfield">
                    <input type="text">
                    <label>input</label>
                </div>
                <div class="textfield">
