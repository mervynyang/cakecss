cakecss简介

cakecss是基于移动端开发的轻量级material design的css库。可通过bower安装。或者直接下载zip文件。

cakecss主要的样式定义为：类名 + 辅助样式 + 辅助样式，比如： <button class="button success lg">button</button>。可以多个辅助样式一起使用，也能单独使用。


tips：在cakecss中，1rem约等于10px。建议所有单位都用rem，因为cakecss rem是根据媒体查询计算出来的。

                
                    bower install onecake
